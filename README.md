# benchat
A bench tool to stress test a chat channel

## Setup

Make sure to have `nvm` and to activate the version specified in the .nvmrc files.

## Running the benchmark

```bash
yarn

./bench.sh --apiKey=... --apiSecret=... --channelType=... --channelID=...
```

## Usage

```bash
./bench.sh -h
```
